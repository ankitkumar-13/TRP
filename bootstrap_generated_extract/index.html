Travel Route Planner

Based on your requirements you can get a path that has minimum cost of travel (distance), across  a number of given location.

Try out the Solution

Info

Based on traveling salesman problem, the objective of this project is to simulate a solution in various ways.

Implementations

The current solutions is supported by the brute-force and the dynamic programming based approach.

This problem is solved in exponential time complexity and there currently is no way to bring it down to a polynomial one.

Technologies Used

Different set of software used to develop.

Bootstrap 5

Bootstrap 5 is used to design the front end of the website for a fast and robust development of the project.

Flask

Flask is the backend support for deploying the website and managing the internal code and template implementation.

Leaflet

A service providing interactive map service for getting a visual representation of the result.
